# MERN Stack Full Course

A complete MERN Stack course project covering:

- MongoDB
- Express.js
- React.js
- Node.js
- REST APIs
- Authentication
- CRUD Operations
- JWT Authentication
- File Upload
- State Management
- Full Stack Application Development

## Tech Stack

### Frontend
- React.js
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## Features

- User Authentication
- Login & Registration
- CRUD Operations
- REST API Integration
- Protected Routes
- File Upload
- Responsive Design

## Installation

### Clone Repository

```bash
git clone https://github.com/your-username/mern-stack-full-course.git
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```
